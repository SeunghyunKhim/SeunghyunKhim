- 꼭 필요했던 기능이었는데 구현 성공해서 기분이 좋다.


예를 들어.

### 일기 모음
카테고리 항목에 journal 이 들어간 노트 리스트 출력.

<ul>
  {% for note in site.notes %}
    {% if "journal" == note.category %}
        <li>
       {{ note.last_modified_at | date: "%Y-%m-%d" }} - <a href="{{ note.url }}">{{ note.title }}</a>
        </li>
    {% endif %}
  {% endfor %}
</ul>
